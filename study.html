Live Study Sessions

Weekly. Live. Together.

Every week the community gets together for a live session. Solver work, spot analysis, concept deep-dives — the kind of session you can't replicate alone.

Session Type
Solver Deep Dives

We pull up spots in a solver, run the ranges, and break down why the equilibrium looks the way it does. Theory you can actually apply.

Session Type
Live Play Review

Screen-share sessions reviewing real live play footage. We pause, discuss, and dissect decisions in real time as a group.

Session Type
Concept Workshops

One concept, one session, total focus. Topics like bet sizing tells, 3-bet ranges by position, GTO vs. exploit adjustments, and more.

Session Type
Range Drilling

Flashcard-style range work as a group. Build the muscle memory for preflop and postflop ranges that actually hold up under pressure.
